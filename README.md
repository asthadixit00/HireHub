# HireHub
Job Portal built with MongoDB, Express, React, Node.js

A Job Portal built with the MERN Stack.
Candidates can browse and apply for jobs. Recruiters can post
and manage listings. Admins oversee the platform.

## Tech Stack
- **Frontend:** React.js, Vite, Axios, React Router
- **Backend:** Node.js, Express.js, REST API
- **Database:** MongoDB, Mongoose
- **Auth:** JWT (Access + Refresh Tokens), bcrypt
- **File Upload:** Multer + Cloudinary
- **Security:** Helmet, CORS, express-rate-limit
- **Deployment:** Render (Backend) + Vercel (Frontend) + MongoDB Atlas

## Features
- [ ] User Authentication (Register/Login/Logout)
- [ ] Role-Based Access (Candidate / Recruiter / Admin)
- [ ] Job CRUD Operations
- [ ] Job Application & Status Tracking
- [ ] Resume Upload
